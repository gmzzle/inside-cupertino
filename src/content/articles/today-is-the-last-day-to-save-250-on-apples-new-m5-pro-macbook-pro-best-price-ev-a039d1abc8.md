[AppleInsider](https://appleinsider.com/articles/26/04/29/today-is-the-last-day-to-save-250-on-apples-new-m5-pro-macbook-pro-best-price-ever?utm_source=rss) reported that B&H Photo is dropping the 14-inch M5 Pro MacBook Pro to $1,949 today through a stacked discount - $200 instant, $50 at checkout. That's $250 off a machine that normally starts at $2,199, and it's the lowest price since the M5 models launched earlier this year.

What makes this worth paying attention to: Apple rarely discounts Pro-tier Macs this early in their cycle. We're only a few months past release, and the M5 Pro is the model most people should actually buy if they're spending this much. The base M5 is fine for casual work, but the Pro chip gets you the extra GPU cores and memory bandwidth that justify the MacBook Pro chassis in the first place. At $1,949, you're basically getting Pro performance at what used to be base-model pricing a generation ago.

The catch, if you want to call it one, is that this is B&H doing the discounting, not Apple. That means no flexibility on AppleCare timing if you prefer adding it later, and you're buying the standard config - 24GB RAM, 512GB storage. For a lot of people, that's plenty. For anyone doing serious video work or running multiple VMs, you were always going to configure up anyway, and this deal doesn't apply to custom builds.

Apple's 2026 MacBook Pro lineup has been weirdly stable on pricing compared to the volatility we saw last year with M4 models. The M5 and M5 Pro arrived without the usual $100-200 markup that sometimes sneaks in during a redesign year, which made them decent value at launch. At $250 off, the M5 Pro is now priced where the M4 Pro should have been all along.

If you've been waiting for the right moment to jump on a 14-inch Pro, this is it. Not because of artificial urgency - though yes, the deal ends today - but because $1,949 is legitimately the floor for this config, and it's unclear when we'll see it again.
